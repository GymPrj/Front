# Gym

## 🛠 사용기술

- React (CRA)
- 스타일링 : styled-components, Mui(material-ui)
- DB : spring
- 상태관리 : redux + redux-sage + redux-persist
- 개발 환경 : eslint-config-airbnb, prettier

## ✨ 주요기능

- 인증 (로그인, 회원가입, 로그아웃)
- 헬스장 (조회)
- 트레이너 (등록, 조회, 코멘트)

## ⛑ Structure

```
src

```
